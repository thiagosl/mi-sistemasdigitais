# mi-sistemasdigitais
O simulador deve ser compilado e executado utilizando os seguintes comandos, em plataformas UNIX e Windows:



$ gcc -o simulator simulator.c

$ ./simulator result.bin

O primeiro comando compila o código do simulador e o segundo executa o simulador utilizando como arquivo de entrada o arquivo result.bin obtido no montador. O simulador gera como saída o arquivo result.txt que irá conter os últimos valores armazenados na memória de dados do processador.
